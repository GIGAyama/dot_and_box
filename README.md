# ドットアンドボックス (Dot and Box)

Google Apps Script (GAS) 上で動作する、2人対戦用のシンプルな陣取りパズルゲームです。点（ドット）と点を結ぶ線を交互に引き、最終的に多くの正方形（ボックス）を完成させたプレイヤーが勝利となります。

## 特徴

* **直感的なUI**: 丸みのあるフォント（M PLUS Rounded 1c）とパステルカラーを採用した、親しみやすいデザイン。
* **カスタマイズ可能な盤面**: 3x3から6x6まで、プレイヤーの好みに合わせて盤面のサイズを選択可能。
* **ゲーム進行機能**:
    * **一手戻す**: 間違えて線を引いた場合でも、直前の状態に復元可能。
    * **リアルタイムスコア**: プレイヤーごとの獲得したボックス数が常に表示されます。
    * **勝利演出**: ゲーム終了時には、勝者を祝う紙吹雪のアニメーションが表示されます。
* **レスポンシブ対応**: スマートフォンやPCなど、異なる画面サイズでも最適に表示されます。

## 技術スタック

* **サーバーサイド**: Google Apps Script (JavaScript)
* **クライアントサイド**: HTML5, CSS3, JavaScript (Vanilla JS)
* **フォント**: Google Fonts (M PLUS Rounded 1c)

## ファイル構成

* `code.gs`: Webアプリとして公開するためのサーバーサイド処理。HTMLテンプレートの配信とメタタグの設定を行います。
* `index.html`: ゲームの構造（HTML）、デザイン（CSS）、およびゲームロジック（JavaScript）のすべてが含まれています。

## 遊び方

1.  **ゲーム開始**: 盤面のサイズ（ドットの数）を選択し、「ゲーム開始」ボタンを押します。
2.  **線を引く**: ドットの間のグレーの領域をクリックして、垂直または水平の線を引きます。
3.  **ボックスの獲得**: 4本目の線を引いて正方形を完成させると、そのボックスは自分の色になり、スコアが加算されます。ボックスを完成させたプレイヤーは、続けてもう一度線を引くことができます。
4.  **交代**: ボックスが完成しなかった場合、相手のターンに移ります。
5.  **終了**: すべてのボックスが埋まるとゲーム終了です。スコアの高いプレイヤーが勝ちとなります。

## インストール・デプロイ

1.  Google Apps Script プロジェクトを作成します。
2.  `code.gs` と `index.html` の内容をそれぞれのファイルにコピー＆ペーストします。
3.  「デプロイ」>「新しいデプロイ」を選択し、種類を「ウェブアプリ」として公開します。
4.  発行されたURLにアクセスすることでゲームをプレイできます。
